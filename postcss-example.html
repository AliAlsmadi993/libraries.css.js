مثال PostCSS مع Autoprefixer

يُظهر المثال الفرق بين الكود الأصلي وكود CSS المعالج (الذي تمت معالجته خارجيًا باستخدام PostCSS وAutoprefixer).

PostCSS Box

الكود الأصلي (CSS):

#postcssBox {
  background-color: #2ecc71;
  padding: 20px;
  border-radius: 8px;
  color: white;
  text-align: center;
  user-select: none;
  transition: transform 0.3s;
}
#postcssBox:hover {
  transform: scale(1.1);
}

الكود المعالج (Processed CSS):

#postcssBox {
  background-color: #2ecc71;
  padding: 20px;
  border-radius: 8px;
  color: white;
  text-align: center;
  -webkit-user-select: none;
  -moz-user-select: none;
  -ms-user-select: none;
  user-select: none;
  transition: transform 0.3s;
}
#postcssBox:hover {
  transform: scale(1.1);
}

ملاحظة: تم معالجة الكود باستخدام PostCSS مع Autoprefixer خارج المتصفح، ثم تم تضمينه مباشرةً في الصفحة.
